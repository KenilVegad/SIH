# 🏏 Cricket Ground Booking System

A full-stack web application developed using Spring Boot that simplifies the process of booking cricket grounds online. The system provides secure role-based access for users and administrators, enabling efficient management of ground reservations, approvals, payments, and booking history.

---

## 📖 Overview

The Cricket Ground Booking System digitizes the traditional ground reservation process by allowing users to browse available cricket grounds, reserve time slots, and track booking status. Administrators can manage grounds, approve or reject bookings, monitor users, and oversee booking activities through a centralized dashboard.

The application follows the Spring Boot MVC architecture with secure authentication and persistent database storage.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/KenilVegad/springboot_criket_booking.git
```

### Navigate

```bash

```

### Run

```bash

```

